About this API

API description and instructions

This is a public API containing the catalog of The Cure's live and studio albums. The catalog includes album title, release year, label, and album category (studio, live, compilation, or extended play). Users can access the API directly or by using the search field in the Search the Catalog section of this page.

Data for individual albums may be accessed by adding the album name to the end of the API string after a forward slash. For example, to return the data associated with "Faith", the user would add /faith to the end of the URL; cure-discography.herokuapp.com/api becomes https://cure-discography.herokuapp.com/api/faith.

API Link
